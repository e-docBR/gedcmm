# Logo e-docBR - Instruções

## Logo Principal do Projeto

A logo oficial da e-docBR deve ser colocada neste diretório:
- **Arquivo**: `edocbr-logo.png`
- **Dimensões recomendadas**: 800x200px (ou proporção similar)
- **Formato**: PNG com fundo transparente

## Uso da Logo

A logo é utilizada em:
1. README.md (topo do repositório)
2. Documentação oficial (docs/)
3. Interface web (frontend)
4. Apresentações e materiais de marketing

## Arquivos de Logo

Organize os seguintes arquivos neste diretório:

```
assets/images/
├── edocbr-logo.png          # Logo principal (fundo escuro)
├── edocbr-logo-light.png    # Logo para fundos claros
├── edocbr-logo-white.png    # Logo branca (para fundos escuros)
├── edocbr-icon.png          # Ícone quadrado (512x512px)
├── edocbr-favicon.ico       # Favicon para web
└── alfresco-integration.png # Diagrama de integração
```

## Cores da Marca

Based on the logo:
- **Azul Principal**: #1B4F8C (rgb(27, 79, 140))
- **Vermelho BR**: #E30613 (rgb(227, 6, 19))
- **Branco**: #FFFFFF
- **Cinza Escuro**: #2C3E50

## Diretrizes de Uso

1. Sempre manter proporção original
2. Não distorcer ou rotacionar
3. Manter espaço mínimo ao redor (padding)
4. Usar versão apropriada para o fundo (claro/escuro)
